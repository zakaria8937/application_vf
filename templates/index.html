Équations d'État

Les quatre modèles thermodynamiques implémentés dans EOS Compare

Van der Waals

1873
Classique
$$P = \frac{RT}{V_m - b} - \frac{a}{V_m^2}$$
Paramètres
$$a = \frac{27\,R^2\,T_c^2}{64\,P_c}$$
$$b = \frac{R\,T_c}{8\,P_c}$$
Où $T_c$ est la température critique, $P_c$ la pression critique, et $R$ la constante des gaz parfaits.

Redlich-Kwong

1949
Améliorée
$$P = \frac{RT}{V_m - b} - \frac{a}{T^{0.5}\,V_m(V_m + b)}$$
Paramètres
$$a = 0.42748\;\frac{R^2\,T_c^{2.5}}{P_c}$$
$$b = 0.08664\;\frac{R\,T_c}{P_c}$$

Soave-Redlich-Kwong

1972
Populaire
$$P = \frac{RT}{V_m - b} - \frac{a(T)}{V_m(V_m + b)}$$
Paramètres
$$a_c = 0.42748\;\frac{R^2\,T_c^2}{P_c}$$
$$b = 0.08664\;\frac{R\,T_c}{P_c}$$
$$a(T) = a_c\,\alpha(T_r,\,\omega)$$
$$\alpha = \left[1 + m\left(1 - T_r^{0.5}\right)\right]^2$$
Avec : $m = 0.480 + 1.574\,\omega - 0.176\,\omega^2$

Peng-Robinson

1976
Référence
$$P = \frac{RT}{V_m - b} - \frac{a(T)}{V_m^2 + 2bV_m - b^2}$$
Paramètres
$$a_c = 0.45724\;\frac{R^2\,T_c^2}{P_c}$$
$$b = 0.07780\;\frac{R\,T_c}{P_c}$$
$$a(T) = a_c\,\alpha(T_r,\,\omega)$$
$$\alpha = \left[1 + \kappa\left(1 - T_r^{0.5}\right)\right]^2$$
Avec : $\kappa = 0.37464 + 1.54226\,\omega - 0.26992\,\omega^2$
